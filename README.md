# React Auth & Role-Based Routing

A React authentication system with role-based access using React, Vite, and MUI.

## Overview

This app provides login, registration, and separate dashboards for users and admins.

Access to routes is controlled based on authentication and user roles (`user`, `admin`).

Works on both **mobile** and **desktop** screens.

## Features

- Login and registration system  
- User dashboard  
- Admin dashboard  
- Protected routes  
- Role-based access control  
- Automatic redirection for unauthorized access  

## Built With

- React  
- Vite  
- React Router DOM  
- Material UI (MUI)  
- Context API  
- JavaScript  

## Getting Started

### Clone the repository

```bash
  git clone https://github.com/aye2-win/auth-dashboard.git
```

Go to the project directory

```bash
  cd auth-dashboard
```

Install dependencies

```bash
  npm install
```

Run the app

```bash
  npm run dev
```
